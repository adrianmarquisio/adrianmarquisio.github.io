El Sitio también recopila información de identificación personal como direcciones de Protocolo de Internet (IP) para usuarios registrados o cuando se completa y envía un formulario. Este Sitio solo divulga las direcciones IP de los usuarios registrados o no bajo las mismas circunstancias que usa y divulga información de identificación personal como se describe a continuación. 3. Este Sitio podrá utilizar también esa información para propósitos estadísticos o de estudio del comportamiento de los visitantes. Los visitantes siempre pueden negarse a proporcionar información de identificación personal, con la advertencia de que puede evitar que participen en ciertas actividades relacionadas con el Sitio. 4.
